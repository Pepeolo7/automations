# AI Consulting — Templates & Demos

Prompts de sistema, workflows n8n e templates prontos para implementação em clientes.
Organizados por departamento, focados em soluções **disruptivas** — não o básico que todos fazem.

## Estrutura

| Pasta | Departamento | Templates |
|---|---|---|
| `suporte-disruptivo/` | Suporte ao Cliente | Deteção de emoção, contacto proativo, análise pós-interação |
| `comercial/` | Vendas | Coaching AI, auto-proposta, previsão churn, mystery shopper |
| `rh/` | Recursos Humanos | Previsão turnover, onboarding autónomo, avaliação contínua |
| `juridico/` | Jurídico/Compliance | Monitor regulatório, compliance contínua, due diligence |
| `financas/` | Finanças | Previsão cash flow, deteção fraude, relatórios narrativos |
| `operacoes/` | Operações/Logística | Reposição inteligente, rotas, manutenção preditiva |
| `marketing/` | Marketing | Segmentação, pricing dinâmico, sentimento de marca |
| `gestao-executiva/` | Direção | Briefing matinal, simulação cenários, prep reuniões |
| `ti/` | TI/Infraestrutura | Auto-healing, segurança preditiva, gestão licenças |
| `formacao/` | Formação | Mentor AI, simulação prática, knowledge capture |

## Como Usar

- **Prompts**: Copiar diretamente para OpenAI/Anthropic API, Intercom Fin, ou qualquer AI agent
- **Workflows n8n**: Importar via `Settings > Import Workflow` no n8n
- **Adaptar**: Todos os templates têm placeholders `[EMPRESA]`, `[SETOR]`, `[IDIOMA]` para personalização
