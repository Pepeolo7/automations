# Comercial / Vendas — AI Disruptiva

Soluções que transformam a equipa comercial: de reativa para preditiva, de manual para assistida por AI.

## Templates

| Ficheiro | Tipo | Descrição |
|---|---|---|
| `sales-coaching-prompt.md` | Prompt | AI analisa reuniões de vendas e dá feedback/coaching ao vendedor |
| `auto-proposal-prompt.md` | Prompt | Gera proposta comercial a partir de notas da reunião |
| `churn-prediction-prompt.md` | Prompt | Analisa sinais de churn em clientes existentes |
| `mystery-shopper-prompt.md` | Prompt | AI simula cliente e avalia qualidade do atendimento comercial |
| `churn-prediction-workflow.json` | n8n | Workflow: analisa dados CRM → identifica risco → alerta equipa |

## Impacto Esperado

- **Coaching AI**: +15-25% taxa de conversão (feedback em cada interação, não apenas trimestral)
- **Auto-proposta**: -80% tempo de criação de propostas (de 2h para 15min)
- **Previsão churn**: -20-35% churn (intervenção 30-90 dias antes da saída)
- **Mystery shopper**: Avaliação contínua sem custos de empresa externa
